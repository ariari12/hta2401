주제: 중고 IT도서 판매 사이트

<메인화면>
- 회원가입
- 로그인
- ID찾기
- PW찾기
- 종료

<로그인 화면>
- 거래 -> <거래>로 이동
- 마이페이지 -> <member>로 이동

- 로그아웃 -> <메인화면>으로 이동

<member>
  - 사용자
    - 마이페이지 - member 속성들 출력
      - 수정
      - 거래 내역 -> 거래 내역 페이지로
      - 탈퇴(삭제)
  - 관리자
   - 계좌 관리
     - 전체 계좌 내역
       - 최신순
       - 날짜순
       {
          - 상세 보기
       }
     - 특정 사용자 입출금 내역
     - 거래 계좌 입금 내역
     - 거래 계좌 출금 내역
     {
       - 상세 보기
     }
   - 전체 거래 내역 -> 전체 거래 내역 페이지로
    - 회원 관리(삭제, 수정)
      - 회원 삭제
      - 회원 정보 수정
      - 회원의 거래 내역 -> 해당 회원의 거래 내역 페이지로

<거래>
   - 판매
     - 내가 등록한 거래글 조회(R)
       - 등록한 거래글 수정
       - 등록한 거래글을 삭제
     - 거래글 등록
       - 거래할 도서 등록(C) - 도서 테이블에 정보 insert
       - 거래 정보 등록(C) -  거래글 테이블에 insert
  - 구매
     - 전체 거래글 목록 조회(R)
       - 검색
       - 조회수
       - 등록순
         - 거래글 상세 조회
           - 중고 도서 구매(거래글 번호)(C)
             - 마일리지 사용?(y/n)
               - n: 마일리지 사용 X
               - y: 마일리지 사용 구매
	  {
                 - 마일리지 제외 구매 금액의 0.5% 마일리지 적립
	     - 계좌에 입금 내역 insert
	  }
            - 불량 판매자 신고(C)
             - 해당 사용자가 신고 횟수가 3회일 경우, 해당 사용자 정지(D)

<거래 내역>
- 전체 거래 내역(관리자만 조회 가능)
  - 상세 보기
- 특정 회원의 거래 내역
   - 특정 회원의 구매 횟수/판매 횟수 cnt
   - 상세보기
      (- 구매 내역만
         - 상세보기
      - 판매 내역만
      - 상세 보기)
      - 판매자
         - 도서 정보 -> 도서 테이블에서 해당 도서 정보 한 줄로 출력
         - 도서 배송 상태(물건 준비중/발송)
           - Default: 물건 준비중
           - 발송완료(책 배송 상태 발송으로 변경)
             - 송장 번호
         - 거래 금액
         - 거래 일자
     - 구매자
         - 도서 정보 -> 도서 테이블의 도서 정보 + 도서 상태 한 줄로 출력
         - 도서 배송 상태
         - 구매 취소(물건 준비중 상태에만 가능)
         - 구매확정(도서 배송 상태 배송 완료로 변경)(U)
           - 계좌에서 거래금액 * 0.9만큼 판매자에게 전달
             -> 해당 거래글 판매자 계좌로
         - 구매 금액
         - 구매 일자

<qna (건의 사용자-관리자)>
  - 사용자
    - 질문 등록
    - 나의 질문 내역(답변 상태(처리중/답변완료) 출력)
      - 상세 보기
        - 질문 수정
        - 질문 삭제
    - 모든 질문 내역(다른 사용자 질문 포함)
      - 상세 보기
  - 관리자
    - 질문 내역
       - 전체 질문 내역
       - 답변 미완료 질문
       - 답변 완료 질문
       {
         - 상세 보기
           - 답변 등록(답변 내용 / 답변 일자 update)
           - 질문 삭제
       }

<퀴즈 이벤트 관리(관리자만)>
- 퀴즈 내역
  - 전체 퀴즈 내역
  - 진행 중인 퀴즈 내역
  - 종료된 퀴즈 내역
   {
     - 상세 보기
       - 퀴즈 수정
       - 퀴즈 삭제
   }
- 퀴즈 등록

<퀴즈 이벤트 참여(사용자만)>
- 퀴즈 내역
  - 전체 퀴즈 내역
  - 진행 중인 퀴즈 내역
    {
      - 상세 보기
        - 퀴즈 참여(이미 참여했을 경우/기간 외일 경우 안됨)
          - 퀴즈 결과에 따라 member 테이블의 마일리지 +
    }
- 내가 참여한 퀴즈 내역
     - 상세 보기

<룰렛 관리(관리자만)>
- 경품 목록
  - 경품 수정
  - 경품 삭제
    {
      -100 - (현재 모든 경품 확률) = 1포인트 경품이 되도록 update
      - 모든 경품 확률의 합 <= 100, 위반일 경우 롤백
     }
- 경품 추가
  - 모든 경품 확률의 합 <= 100, 위반일 경우 롤백
- 모든 사용자의 룰렛 참여 내역
  - 상세 보기

<룰렛 참여(사용자만)>
- 룰렛 참여
  - 내역에 해당 사용자가 당일 기참여시 재참여 안됨
    - 당일 내역 출력
  - 당일 첫참여시 확률에 따라 마일리지 지급
    - member 테이블에 마일리지에 지급 포인트만큼 +

CRUD
- 업적
  - 사용자
    (- 출석 누적 일수에 따른 업적)
    - 물건 판매 갯수에 따른 업적
    - 물건 구매 갯수에 따른 업적
    => 업적 획득에 따라 마일리지 누적

- 마일리지 샵
  - 상설
   - 3% 할인
   - 5% 할인
   - 10% 할인
   - 100명 선착순 기프티콘(사이트 외부에서 사용)
  - 기간 한정
    - 1주년 기념 15% 할인
    - 크리스마스 기념 15% 할인
    - 신학기 기념 15% 할인
    - 생일 기념 10% 할인

- 마일리지 샵 관리(관리자 메뉴)
  - 아이템 등록
  - 아이템 조회
    - 아이템 수정
    - 아이템 삭제
